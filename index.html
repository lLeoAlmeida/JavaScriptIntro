Introdução ao Javascript



Olá xuxuzinhos do Tio Léo! Sejam bem-vindos à nossa primeira aula sobre do JavaScript, hoje vamos aprender os conceitos básicos dessa poderosa linguagem de programação e como utilizá-la para criar interatividade em suas páginas web.


Antes de colocarmos a mão na massa, vamos ver um pouco sobre a história do nosso amigão JavaScript.
JavaScript é a linguagem de programação Web, em que ampla maioria dos sites e navegadores modernos utilizam ele nativamente, tornando assim a linguagem de programação mais onipresente da história. O nome “JavaScript” é um pouco enganoso, pois o JavaScript é completamente diferente da linguagem de programação Java e também já deixou para trás suas raízes como linguagem de script há muito tempo, tornando-se uma linguagem de uso geral robusta e eficiente.
O JavaScript é uma linguagem case sensitive, ou seja, diferencia letras maiúsculas de minúsculas, isso significa que palavras-chave, variáveis, nomes de função e outros identificadores da linguagem sempre devem ser digitados com a composição compatível de letras, Por exemplo o comando while, deve ser digitada como “while” e não como “While” ou “WHILE".

Incorporando JavaScript no HTML

Para começar a programar em JavaScript, você pode incorporar seu código diretamente em um arquivo HTML usando a tag <script> no final do código, antes de fechar o </body> 
            
              <!DOCTYPE html>
              <html>
              <head>
                <title>Minha Primeira Página JavaScript</title>
              </head>

              <body>
                <h1>Hello World in JavaScript</h1>



                <script src=" "> </script>
              </body>
              </html>

Declaração de variaveis em JavaScript

No JavaScript assim como toda linguagem de programação (Lembrando, que HTML e CSS NÃO SÃO LINGUAGENS DE PROGRAMAÇÃO, elas são linguagens de marcação e estilização respectivamente).

Ele possui variáveis que são usadas para armazenar valores, podemos declarar uma variável usando a palavra-chave var, let ou const e seguida pelo nome da variável e, opcionalmente, atribuir um valor a ela. Existem vários tipos de dados em JavaScript, incluindo números, strings, booleanos e arrays. 

          
                var nome = "Léo";
                let idade = 29;
                let numeros = [1, 2, 3];
                const pi = 3.14;

Operadores

JavaScript possui diversos operadores que permitem realizar cálculos e manipulações de dados. 

            
              let soma = 10 + 5; // soma = 15
              let subtracao = 10 - 5; // subtracao = 5
              let multiplicacao = 10 * 5; // multiplicacao = 50
              let divisao = 10 / 5; // divisao = 2
              let modulo = 10 % 3; // modulo = 1 (resto da divisão)

Function

Uma função é um bloco de código que executa uma tarefa específica. A função é definida usando a palavra-chave function, seguida pelo nome da função, parênteses e um bloco de código entre chaves. Por exemplo, aqui está uma função chamada HelloWorld que exibe uma mensagem de boas-vindas: 

        
          function HelloWorld() {
            console.log("Hello World!");
          }

Quando você quer executar o código dentro da função, você a "chama" pelo nome, seguido por parênteses vazios, como este: 

      
        HelloWorld();

Exibindo uma mensagem no HTML usando JavaScript

Para exibir uma mensagem no HTML usando JavaScript, podemos usar o método document.write() manipulando o conteúdo de um elemento HTML existente.
Antes de utilizar o método document.write() , você precisa ter um elemento HTML no qual aonde você deseja exibir a mensagem. Por exemplo, exibir uma mensagem dentro de uma tag <p > com um id "mensagem" atribuido a ele: 

      
        <!---DOCUMENTO HTML--->

        <!DOCTYPE html>
        <html>
        <head>
          <title>Exibindo uma mensagem no HTML usando JavaScript</title>
        </head>

        <body>
          <p  id="mensagem">
          

          </p>



          <script src="script.js"> </script>
        </body>
        </html>

Agora, você pode usar JavaScript para obter o elemento pelo ID e atribuir um novo conteúdo a ele. 

      
        //Documento script.js

        let mensagemElement = document.getElementById("mensagem");
        mensagemElement.innerHTML = "Olá, mundo!";

Dessa forma, o texto "Olá, mundo!" será exibido dentro do elemento <p> com a id "mensagem".

DOM (Document Object Model)

Básicamente, o DOM é o responsavel que nos permite que os scripts em JavaScript interajam com os elementos de dentro de uma página web. Como feito no exemplo acima, em que inserimos uma mensagem que estava no JavaScript diretametente dentro do elemento <p> no HTML.
